# leetcode

## Find Pivot Index

### 풀이

좌측과 우측의 합계를 비교하며, 같아지는 순간이 있는지 확인한다!

**코드**

```js
var pivotIndex = function (nums) {
  let rightSum = nums.reduce((acc, num) => acc + num, 0);
  let leftSum = 0;
  for (let i = 0; i < nums.length; i++) {
    rightSum -= nums[i];
    if (leftSum === rightSum) return i;
    leftSum += nums[i];
  }
  return -1;
};
```
