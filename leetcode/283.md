# leetcode

## Move Zeroes

### 풀이

값이 0인 요소만 배열의 끝으로 이동한다.

**숫자를 왼쪽으로 이동**

```js
var moveZeroes = function (nums) {
  let i = 0;
  for (let j = 0; j < nums.length; j++) {
    if (nums[j] !== 0) {
      nums[i] = nums[j];
      i++;
    }
  }
  for (; i < nums.length; i++) {
    nums[i] = 0;
  }
};
```

**sort() 적용**

```js
var moveZeroes = function (nums) {
  return nums.sort((a, b) => (a === 0 ? 1 : b === 0 ? -1 : 0));
};
```
